# Phase 0 — RUN/WALK (Beginner Foundation)

## When
- `poziom: beginner` in profile.md
- Person cannot yet run 20–30 minutes continuously without stopping
- Returning after long break (6+ months), treat as beginner

## Goal
Build the habit of running and develop aerobic base from scratch.
**Exit criterion: run 30 minutes continuously at conversational pace (can speak in full sentences).**
After that → graduate to Phase I.

## Duration
8–10 weeks (can compress to 6 if progressing fast, extend to 12 if needed — no rush)

## Weekly structure
3 sessions per week, with at least 1 rest day between each.

### Weeks 1–2 — Mostly walking
- 5 min walk warm-up
- 8× (1 min run / 2 min walk)
- 5 min walk cool-down
- Total: ~35 min

### Weeks 3–4 — Equal intervals
- 5 min walk warm-up
- 6× (2 min run / 2 min walk)
- 5 min walk cool-down
- Total: ~35 min

### Weeks 5–6 — Run-dominant
- 5 min walk warm-up
- 5× (3 min run / 1 min walk)
- 5 min walk cool-down
- Total: ~35 min

### Weeks 7–8 — Continuous runs
- 5 min walk warm-up
- 20 min continuous run
- 5 min walk cool-down
- Week 8: push to 25 min continuous

### Weeks 9–10 — Graduation
- 30 min continuous run × 3/week
- If all 3 sessions completed without stopping → **graduate to Phase I**

## Pace guidance
**Run at a pace where you can speak 2–3 words at a time.** If gasping → slow down or walk.
No GPS pacing targets. Heart rate if available: stay under 75% max HR.
Formula for max HR estimate: 220 − age.

## Avoid
- Skipping rest days (injury risk is highest in weeks 1–4)
- Running through shin or knee pain (soreness is OK, sharp pain is not)
- Comparing pace to others — this phase is about time on feet, not speed

## Footwear check
First session: assess shoes. Running in wrong shoes is the #1 beginner injury cause.
Recommend: visit a running store for gait analysis if available.

## Progress markers
- Each week feels slightly easier at the same effort
- Resting HR trending down over the 10 weeks
- No soreness 48h after runs by week 6

## After Phase 0
→ Graduate to **Phase I — BASE**
→ Do a first VDOT estimate: run a 1km or 1 mile time trial, use calculator at runsmartproject.com
→ Update fitness.md with estimated VDOT and zones
→ Set `poziom: intermediate` in profile.md
